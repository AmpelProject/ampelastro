Astronomical Archives    

A complete archive of Zwicky Transient Facility (ZTF) is maintained at the DESY Zeuthen computer center. This can be used both to access alerts for individual objects as well as to recreate alert streams based on e.g. time or position. These archives are expected to grow to include samples from e.g. the ULTRASAT satellite and the LSST survey carried out at the Vera Rubin Observatory. Base API entry points are listed at ZTF Alert Archive Service

Several convenience tools exist for processing alerts through a local AMPEL environment. A notebook illustrating how to access data for a specific SN can be found here.

To monitor data access, all users need to be registered members of the github AmpelProject. After joining there, a personal archive access token can be obtained from Token Generator.

Catalog matching is critical, and a number of astronomical catalogs have been incorporated into the archive for fast access. These can also be accessed externally: Access catalogue list.
